# Monitor Deportivo — Dashboard en Dash
## Que hace esta app?
Dashboard de monitorizacion de deportistas en tiempo real.
Muestra metricas fisiologicas como frecuencia cardiaca y fatiga.
## Tecnologias usadas
- Python + Dash + Plotly
- BITalino para adquisicion de senales
